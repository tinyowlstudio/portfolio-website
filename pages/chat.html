Meet App

Github Repository

This chat app is a platform for both iOS and Android, and allow users to send messages, images and locations to friends and family. The UI will also be customizable, and data is stored both online and offline.

Features

  • Login with anonymous authorization
  • Chat live with another person
  • Share photos from library/camera
  • Share location
  • Customize UI colors

Tools Used

  • React Native: To build the app structure
  • Expo/Expo Go: For testing
  • Puppeteer: Also for testing
  • Gifted Chat: For data and authorization
  • Android Studio: For serverless toolkit
